# SpyFu

SpyFu is a competitive SEO and PPC intelligence platform that gives marketers and agencies programmatic access to competitor keyword rankings, ad history, backlinks, and domain performance data. The REST API covers eight research domains — Domain Stats, Ad History, PPC Research, SEO Research, Competitors, Kombat, Keyword Research, and Ranking History — indexed against 1.6 trillion search results, 8 billion keywords, and 152 million domains.
